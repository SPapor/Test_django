# SalesFlow CRM

CRM-система для управління клієнтами та угодами. Розроблено у рамках тестового завдання.

## Функціонал

* **Клієнти:** Створення, перегляд, редагування, пошук.
* **Угоди:** 
* Прив'язка до клієнтів. 
* Статуси. 
* Фільтрування.
* **Нотатки:** Можливість залишати коментарі до угод.
* **API:** REST API для інтеграції із зовнішніми сервісами.


## Технічний стек

* **Python 3.12**
* **Django 6.0.1**
* **Django REST Framework**
* **Bootstrap 5**
* **SQLite**
* **Docker & Docker Compose**
* **Unittest**

## Як запустити проект

### Через Docker

1. **Клонуйте репозиторій:** 
``` bash 
git clone https://github.com/SPapor/Test_django.git 
cd Test_django 
````

2. **Запустіть однією командою:** 
``` bash 
docker-compose up --build 
````

3. **Відкрийте у браузері:** [http://127.0.0.1:8000](http://127.0.0.1:8000)
